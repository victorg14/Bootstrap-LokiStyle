Comience con Bootstrap LokiStyle

es un potente conjunto de herramientas frontend repleto de funcionalidades. Crea cualquier proyecto, desde el prototipo hasta la producción, en minutos.

Inicio Rapido

Comienza incluyendo CSS de Bootstrap LokiStyle listos para producción a través de CDN sin necesidad de ningún paso de compilación.

  1. Crea un nuevo index.htmlarchivo en la raíz de tu proyecto. Incluye también la etiqueta para que funcione correctamente en dispositivos móviles. 
                            
                    <!doctype html>
                    <html lang="en">
                        <head>
                            <meta charset="utf-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1">
                            <title>Bootstrap LokiStyle</title>
                        </head>
                        <body>
                            <h1>Hello, world!</h1>
                        </body>
                    </html>
  2. Incluya el CSS. Coloque la etiqueta en el [nombre del archivo] para nuestro CSS
