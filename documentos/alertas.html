Alertas LokiStyle

Las alertas son mensajes rápidos para mostrar información, advertencias, errores o éxitos de una acción. Personalizadas con los colores y estilos mágicos de LokiStyle.

Estructura Básica

                                <div class="alerta alerta-exito">
                                    ¡Éxito! Todo ha salido bien.
                                </div>

Tipos de Alertas

✅ ¡Operación realizada exitosamente!
ℹ️ Esta es una alerta informativa.
⚠️ Cuidado, esta es una advertencia.
❌ ¡Ha ocurrido un error!

¿Cómo funcionan?

  • .alerta: Clase general para todas las alertas.
  • .alerta-exito: Estilo para alertas de éxito (verde).
  • .alerta-info: Estilo para alertas informativas (azul hielo).
  • .alerta-advertencia: Estilo para advertencias (dorado suave).
  • .alerta-error: Estilo para errores (morado mágico).
